--- layout: post title: My dark color theme for NetBeans --- For quite some time, I've been really frustrated with the look and feel of my source editor in NetBeans. I really liked the color schemas I have been using in my vim work flow. In fact it was probably the only thing (perhaps besides speed) that kept me using vim instead of NetBeans. I didn't settle with themes such as Aloha or Dark Pastels, and tried to "port" Nick Moffitt's Railscasts vim theme into NetBeans. You can see the final outcome on the screenshot above. I would gladly package it as a NBM file, however I don't know how one could build a NBM file (yet :-)). The bottom line is, working with NetBeans is a bit more enjoyable now.
